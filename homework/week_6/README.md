## Question 2. Yellow November 2025

Read the November 2025 Yellow into a Spark Dataframe. Repartition the Dataframe to 4 partitions and save it to parquet. What is the average size of the Parquet Files that were created (in MB)?

**Answer:** 25MB

Using `df.repartition(4).write.mode("overwrite").parquet()`, the data is split into 4 files. Checking the file system reveals each partition is approximately 24.42 MB, which most closely matches 25MB.

## Question 3. Count records

How many taxi trips were there on the 15th of November? Consider only trips that started on the 15th of November.

**Answer:** 162,604

Calculated by filtering the DataFrame where the date part of `tpep_pickup_datetime` equals `2025-11-15` and executing the `.count()` action.

## Question 4. Longest trip

What is the length of the longest trip in the dataset in hours?

**Answer:** 90.6

Calculated by extracting the UNIX timestamp difference between `tpep_dropoff_datetime` and `tpep_pickup_datetime`, dividing by 3600 to convert to hours, and finding the maximum value (90.65 hours).

## Question 5. User Interface

Spark's User Interface which shows the application's dashboard runs on which local port?

**Answer:** 4040

By default, when running a local Spark session, the Spark web UI binds to `http://localhost:4040` to monitor jobs, stages, and executors.

## Question 6. Least frequent pickup location zone

Using the zone lookup data and the Yellow November 2025 data, what is the name of the LEAST frequent pickup location Zone?

**Answer:** Governor's Island/Ellis Island/Liberty Island

Determined by joining the taxi data `PULocationID` with the zone data `LocationID`, grouping by `Zone`, and ordering by the trip count in ascending order. This zone had exactly 1 trip.
